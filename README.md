# 🌱 AgroTech Sustentável

Sistema inteligente de agricultura sustentável com simulação, IA educativa e gamificação para incentivo de práticas ecológicas no campo.

## 📌 Objetivo
Promover a conscientização sobre sustentabilidade no campo por meio da tecnologia, simulações interativas e inteligência artificial educativa.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript
